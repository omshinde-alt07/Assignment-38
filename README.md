# Word2Vec Assignment

This project contains the notebook **word2vec.ipynb** with solutions for NLP embedding tasks.

## File Included

github link : https://github.com/omshinde-alt07/Assignment-38

* `word2vec.ipynb` — Jupyter Notebook containing code, outputs, and explanations.

## Topics Covered

* Training Word2Vec on ShopSense reviews
* Polysemy of the word *cheap*
* Cosine similarity comparisons
* Word sense disambiguation using anchor vectors
* Window size comparison (`2` vs `10`)
* BOW vs TF-IDF vs Word2Vec vs Sentence-BERT
* Semantic similarity analysis

## Requirements

Install dependencies before running:

```bash
pip install pandas numpy scikit-learn gensim sentence-transformers jupyter
```

## How to Run

1. Open Jupyter Notebook.
2. Load `word2vec.ipynb`.
3. Run cells in order.

## Notes

* If `gensim` fails, use Python 3.12.
* For newer Python versions, some packages may require compatibility fixes.
